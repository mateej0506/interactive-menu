# interactive-menu
Frontend project: Add, view, and delete drinks with descriptions (HTML, CSS, JS)
This is a simple interactive web project that allows users to add drinks, view their descriptions, and remove them from a list. It demonstrates DOM manipulation, event handling, and working with JavaScript objects.

Features

Add a drink by typing in the input field and clicking "Submit" or pressing Enter.
View the drink's description dynamically pulled from a predefined list.
Remove individual drinks with a delete button.
Clear all drinks at once using the "Remove All" button.
Works entirely on the frontend using HTML, CSS, and JavaScript.

Technologies Used
HTML5
CSS3
JavaScript 

How to Use
Type the name of a drink in the input field.
Press "Submit" or hit Enter.
The drink will appear below with its description.
Use the "x" button on each drink to remove it individually, or "Remove All" to clear the list.

Notes
Only drinks included in the predefined list have descriptions.
Any drink not in the list will display "No available description."
